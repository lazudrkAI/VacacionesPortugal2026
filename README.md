# VacacionesPortugal2026

Repositorio que contiene una base de datos estática de vacaciones centrada en el entorno de Viana do Castelo (norte de Portugal).

Qué hay en este repositorio

- Datos estáticos sobre lugares de interés, playas, rutas y actividades en la zona de Viana do Castelo.
- Información sobre alojamientos (hoteles, pensiones, apartamentos) y contactos relevantes.
- Recomendaciones y notas prácticas para planificar una estancia (transporte, gastronomía, mejores épocas, etc.).

Formato de los datos

Los datos se encuentran en archivos estáticos (por ejemplo, JSON, CSV o Markdown) dentro del repositorio para facilitar su consulta y edición manual. Cada archivo está pensado para ser independiente y fácil de leer.

Cómo contribuir

Si quieres añadir o corregir información:

1. Haz un fork del repositorio.
2. Crea una rama nueva para tu cambio.
3. Añade o modifica los archivos de datos.
4. Abre un pull request con una descripción clara de los cambios.

Licencia y uso

Este repositorio está pensado como referencia personal/compartida. Revisa los archivos individuales para ver si hay notas sobre licencias o atribuciones de datos externos.

Contacto

Para preguntas o sugerencias, abre un issue en el repositorio.
